# Medeasy - Calm and Reliable Drug Search App

A modern, responsive React application built with Tailwind CSS, Framer Motion, and Lucide React.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Features

- **Landing Page:** Blurred background with a central logo interaction to proceed.
- **Persistent Sidebar:** Navigation for About Us, Search, Data Sources, and Contact.
- **Drug Search:** Clean, intuitive search interface.
- **Drug Details:** Interactive card-based layout for:
  - Details of tablet contents
  - Indication and contraindication
  - Interaction with other medications
  - Side effects
  - Interaction with food

## Tech Stack

- **Framework:** React (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Routing:** React Router DOM
- **Font:** Inter (Clean and Professional)
